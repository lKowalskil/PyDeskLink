# PyDeskLink

## Overview

PyDeskLink is a cutting-edge remote desktop application designed for system administrators. It offers a user-friendly GUI, enabling efficient monitoring and control of systems remotely. This tool stands out for its comprehensive feature set, ease of use, and robust performance.

## Features

Each feature of PyDeskLink enhances remote system management:

- **System Information**: Provides detailed information about the client's system, aiding in diagnostics and monitoring.
- **CMD**: Execute shell commands remotely, offering the same versatility as local access.
- **File and Directory Discovery**: Easily navigate and manage the client's files and directories.
- **Remote File Copy**: Securely copy files to and from the client system.
- **File Deletion**: Remotely delete files, ensuring data privacy and security.
- **Process Discovery**: Monitor and manage running processes for better system oversight.
- **Input Capture**: Record keystrokes and mouse activity for security auditing.
- **Clipboard Data**: Access and manage the client's clipboard contents.
- **Screenshot Capture**: Take and retrieve screenshots for remote troubleshooting.
- **Screen Capture**: Share the client's screen in real-time for collaborative work or support.
- **Audio Capture**: Record audio for remote monitoring or assistance.
- **Video Capture**: Use the client's webcam to capture video, enhancing remote communication.

## Getting Started

### Prerequisites

- Python 3.11

### Installation

```bash
git clone https://github.com/lKowalskil/PyDeskLink.git
cd PyDeskLink
pip install -r requirements.txt
```
